## 常见解题思路
1. 数组中第一位数作为默认的候选人(cand_num)
2. 依次向后遍历
3. 当出现相同的数(cand_num) 则 投票数 count +1
4. 当出现不同的数 则投票(count_num) 投票数 count -1
5. 当 count ==0 ，则跟换候选人，并 count 重制为 1
6. 遍历完后 count_num 则为最终答案
